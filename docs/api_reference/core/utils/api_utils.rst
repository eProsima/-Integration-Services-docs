Utils
-----

This section of the API reference corresponds to the `include/is/utils <https://github.com/eProsima/Integration-Service/tree/main/core/include/is/utils>`_
folder of the *Integration Service* main repository.

This folder contains the logger tool and the conversion library used for those middlewares that
use static types, such as *ROS 2* and *ROS 1*.
